Node.js

With Node.js you can use the very popular programming language JavaScript to write software. JavaScript is usually used in a browser, but combined with other tools, Node.js allows you to write desktop applications, server applications or even mobile applications

NodeSchool

NodeSchool are open source workshops that teach web software skills. Workshoppers are used as curriculum and mentors help attendees work through the challenges.

Mentors

Format

We use self guided tutorials, or workshoppers, as lesson modules. You can do them on your own, but we encourage you to work together.

Choosing a workshopper

Core workshoppers focus on essential skills for working with Node.js. If you’re new to JavaScript and Node.js, pick these.

Electives deal with more advanced topics. Pick one of these if you already have some experience.

Core workshoppers

If you’re new to JavaScript, start with javascripting

 $ npm install -g javascripting

Or, if you already know JavaScript, but haven’t worked with Node.js before, try out learnyounode

 $ npm install -g learnyounode

And teach yourself some source control skills with git-it

 $ npm install -g git-it

Advanced core workshoppers

Create your first npm module with how-to-npm

 $ npm install -g how-to-npm

Or learn the details of scope, scope chains, closures with scope-chains-closures

 $ npm install -g scope-chains-closures

Recommended electives

If you want to challenge yourself by selecting an elective, we recommend the following workshoppers

 $ npm install -g functional-javascript-workshop

 $ npm install -g async-you

 $ npm install -g promise-it-wont-hurt

 $ npm install -g lololodash

 $ npm install -g expressworks
